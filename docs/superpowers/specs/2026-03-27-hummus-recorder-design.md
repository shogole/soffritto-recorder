# Hummus Recorder — 設計仕様書

**日付**: 2026-03-27
**リポジトリ**: soffritto-recorder
**対象ファイル**: `hummus.html`

## 概要

フムスの製造記録をスマートフォンから入力し、Google Spreadsheetに積み上げるシングルページWebアプリ。`index.html`（ソフリット記録）と同リポジトリに独立ファイルとして追加する。デザイン・操作感はソフリット版を踏襲し、アクセントカラーのみ変更する。

## セクション構成

### 1. ロット情報
| フィールド | 型 | デフォルト |
|-----------|-----|-----------|
| 試作日 | date | 当日 |
| ロット番号 | number | - |
| 天候 | text | - |
| 室温 | text | - |

### 2. 原材料

**吸水計測**
| フィールド | 型 | 初期値 | 備考 |
|-----------|-----|-------|------|
| ひよこ豆（乾燥前）| number (g)、inputmode="decimal" | 空欄 | placeholder="例: 250" |
| ひよこ豆（ゆで後）| number (g)、inputmode="decimal" | `value="400"`（入力済み） | 毎回上書きを想定 |
| 吸水率 | 表示のみ | `--` | 乾燥前 > 0 かつ ゆで後 > 0 のとき計算。それ以外は `--` を表示 |

吸水率 = (ゆで後 − 乾燥前) / 乾燥前 × 100（乾燥前または ゆで後が 0 または未入力の場合は `--`）

**調味料**（すべて `inputmode="decimal"`）
| フィールド | `value`（初期入力済み） |
|-----------|-----------|
| 氷茹で汁 (g) | 70 |
| ガーリックオイル (g) | 5 |
| 塩 (g) | 3 |
| レモン汁 (g) | 10 |
| ねりごま (g) | 10 |

**合計重量**（自動計算・表示のみ）
- 計算対象: ゆで後 + 氷茹で汁 + ガーリックオイル + 塩 + レモン汁 + ねりごま
- アレンジ材料は自由記述のため合計に算入しない

### 3. 工程
| フィールド | 型 | 備考 |
|-----------|-----|------|
| 茹で時間（分）| number、inputmode="decimal" | placeholder="90" |
| 工程メモ | text | placeholder="圧力鍋15分、自然減圧など" |

### 4. 仕上がり評価（5段階チップ選択）

各項目に 1〜5 のチップを横並びで表示。未選択は空文字として送信（必須ではない）。

- なめらかさ：1 / 2 / 3 / 4 / 5
- 塩味：1 / 2 / 3 / 4 / 5
- 酸味：1 / 2 / 3 / 4 / 5
- 全体評価：1 / 2 / 3 / 4 / 5

### 5. アレンジ材料
テキストエリア（自由記述）。placeholder="クミン5g、スモークパプリカ2g など"

### 6. メモ
テキストエリア（自由記述）。placeholder="気づいたこと、改善点など..."

## 自動計算

| 計算項目 | 計算式 | ガード条件 |
|---------|--------|----------|
| 吸水率 | (ゆで後 − 乾燥前) / 乾燥前 × 100 | 乾燥前 > 0 かつ ゆで後 > 0 のときのみ。それ以外 `--` |
| 合計重量 | ゆで後 + 氷茹で汁 + ガーリックオイル + 塩 + レモン汁 + ねりごま | いずれか未入力は 0 として計算 |

## データ送信

- 送信先: Google Apps Script エンドポイント（ソフリットとは別のGAS URL）
- GAS URLは `const GAS_URL = 'YOUR_GAS_URL_HERE';` としてプレースホルダーで実装し、運用時に差し込む
- 方式: `fetch` POST、`mode: 'no-cors'`

**送信JSONキー名一覧**

| キー名 | 内容 |
|--------|------|
| `date` | 試作日 |
| `lotNo` | ロット番号 |
| `weather` | 天候 |
| `roomTemp` | 室温 |
| `dryWeight` | ひよこ豆（乾燥前）g |
| `cookedWeight` | ひよこ豆（ゆで後）g |
| `absorptionRate` | 吸水率 %（乾燥前が0または未入力の場合は `""`） |
| `iceBroth` | 氷茹で汁 g |
| `garlicOil` | ガーリックオイル g |
| `salt` | 塩 g |
| `lemonJuice` | レモン汁 g |
| `tahini` | ねりごま g |
| `totalWeight` | 合計重量 g（全フィールド未入力で合計が0の場合は `""`） |
| `boilTime` | 茹で時間 分 |
| `processMemo` | 工程メモ |
| `smoothness` | なめらかさ（1〜5、未選択は空文字） |
| `saltiness` | 塩味（1〜5、未選択は空文字） |
| `acidity` | 酸味（1〜5、未選択は空文字） |
| `overallRating` | 全体評価（1〜5、未選択は空文字） |
| `arrangement` | アレンジ材料（自由記述） |
| `memo` | メモ |

## デザイン

- ソフリット版と同じレイアウト・コンポーネント（section、chip-group等）
- 背景色: `#1a1a2e`（同じ）
- アクセントカラー: `#d4a853`（ゴールデンベージュ）
  - ソフリット版の `#f5a623` をすべて `#d4a853` に置換（h1、h2、chip.selected、total-barの背景、自動計算表示テキスト等10箇所以上）
  - `total-bar` のテキスト色は `color: #1a1a2e` を維持（コントラスト比 ≥ 4.5:1 を確認済み）
- すべての数値入力フィールドに `inputmode="decimal"` を付与（モバイルキーボード最適化）
- 最大幅500px、モバイルファースト
