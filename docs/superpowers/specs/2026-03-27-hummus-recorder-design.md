# Hummus Recorder — 設計仕様書

**日付**: 2026-03-27
**リポジトリ**: soffritto-recorder
**対象ファイル**: `hummus.html`

---

## 概要

フムスの製造記録をスマートフォンから入力し、Google Spreadsheetに積み上げるシングルページWebアプリ。`index.html`（ソフリット記録）と同リポジトリに独立ファイルとして追加する。デザイン・操作感はソフリット版を踏襲し、アクセントカラーのみフムスらしいクリーム/ベージュ系に変更する。

---

## セクション構成

### 1. ロット情報
| フィールド | 型 | デフォルト |
|-----------|-----|-----------|
| 試作日 | date | 当日 |
| ロット番号 | number | - |
| 天候 | text | - |
| 室温 | text | - |

### 2. 原材料

**吸水計測**
| フィールド | 型 | デフォルト |
|-----------|-----|-----------|
| ひよこ豆（乾燥前）| number (g) | - |
| ひよこ豆（ゆで後）| number (g) | 400 |
| 吸水率（自動計算）| 表示のみ | - |

吸水率 = (ゆで後 − 乾燥前) / 乾燥前 × 100

**調味料**（デフォルト値付き入力）
| フィールド | デフォルト |
|-----------|-----------|
| 氷茹で汁 (g) | 70 |
| ガーリックオイル (g) | 5 |
| 塩 (g) | 3 |
| レモン汁 (g) | 10 |
| ねりごま (g) | 10 |

**合計重量**（自動計算・ゆで後豆＋調味料の合計）

### 3. 工程
| フィールド | 型 |
|-----------|-----|
| 茹で時間（分）| number |
| 工程メモ | text |

### 4. 仕上がり評価（5段階チップ選択）
- なめらかさ：1〜5
- 塩味：1〜5
- 酸味：1〜5
- 全体評価：1〜5

### 5. アレンジ材料
自由記述テキストエリア（例：「クミン5g、パプリカ2g」）

### 6. メモ
自由記述テキストエリア

---

## 自動計算

| 計算項目 | 計算式 |
|---------|--------|
| 吸水率 | (ゆで後 − 乾燥前) / 乾燥前 × 100 |
| 合計重量 | ゆで後 + 氷茹で汁 + ガーリックオイル + 塩 + レモン汁 + ねりごま |

---

## データ送信

- 送信先: Google Apps Script エンドポイント（GAS URL、ソフリットとは別）
- 方式: `fetch` POST、`mode: 'no-cors'`
- 送信データ（JSON）:
  - ロット情報全フィールド
  - 乾燥前・ゆで後・吸水率
  - 調味料各値
  - 合計重量
  - 茹で時間・工程メモ
  - 仕上がり評価4項目
  - アレンジ材料
  - メモ
